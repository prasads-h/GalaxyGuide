Tools:

1) maven
2) junit-4.8.2
3) Java 1.6


Building:
Pre-requisites:
Maven should be setup in eclipse or command prompt

1) Go to Galaxy Guide folder and execute below command
	mvn clean install
2) Go to the target directory GalaxyGuide/target in the command prompt and execute below command	
	java -jar galaxyguide-1.0.jar "specifity input file" "specify out file"
	
	Note: path must be a file not directory
	
3) output file will be generated in the output directory
